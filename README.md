### Passos do projeto

* Construir um gerador de carteiras de Bitcoin;
* Importar a carteira gerada para um software de carteiras;
* Realizar transações de envio e recebimento de Bitcoin.

### Softwares utilizados

* Node.js
* Visual Studio Code
* Electrum
